# Research Lab Manager

A comprehensive database application for managing university research laboratory operations.

## Overview

This project aims to develop a database system to support and streamline operations within a university research laboratory. The system facilitates tracking of:
- Lab members (faculty, students, external collaborators)
- Research projects
- Equipment usage
- Publications
- Funding (grants)
